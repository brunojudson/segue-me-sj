1. Introdução ao Sistema

🎯 Sobre o Sistema Segue-me

O Sistema Segue-me é uma aplicação web completa desenvolvida para gerenciar todos os aspectos relacionados aos Encontros de Jovens, desde o cadastro de participantes até o controle financeiro e geração de relatórios detalhados.

📅

⚙️

💰

Controle Financeiro

Contribuições, vendas e relatórios financeiros

📊

Relatórios Completos

Exportações em PDF e Excel para análises

🔐

Segurança

Controle de acesso e autenticação segura

💡 Dica: Este manual está organizado de forma progressiva. Se você é um novo usuário, recomendamos seguir a ordem das seções. Para usuários experientes, utilize o índice para navegar rapidamente até o tópico desejado.

5. Fluxo de Implantação Inicial

⚠️ ATENÇÃO: Esta seção é fundamental para quem está configurando o sistema pela primeira vez ou migrando dados de outro sistema. Siga rigorosamente a ordem descrita para evitar problemas de integridade.

📋 Ordem Recomendada de Cadastro

O sistema possui relacionamentos entre entidades que devem ser respeitados. Seguir a ordem abaixo evita erros de chave estrangeira e garante a consistência dos dados.

1 Cadastrar TODAS as Pessoas

CRÍTICO Pessoa é a entidade base do sistema

  • Comece por pessoas mais antigas (se estiver migrando dados)
  • Garanta que telefones sejam únicos
  • Valide CPF quando disponível
  • Anexe fotos se disponíveis
💡 Por que de trás para frente?
Ao migrar de planilhas ordenadas, inserir do mais antigo para o mais recente preserva a lógica histórica e facilita identificar duplicidades de telefone.
2 Cadastrar Casais

IMPORTANTE Requer pessoas já cadastradas

  • Selecione Pessoa 1 (esposo/esposa)
  • Selecione Pessoa 2 (esposo/esposa)
  • Informe data de casamento se conhecida
  • Sistema valida unicidade do par
3 Cadastrar Encontro

PADRÃO Use nomenclatura consistente

  • Nome: Segue-me 2026 (padrão recomendado)
  • Datas: Início e fim do evento
  • Local: Endereço completo
  • Capacidade: Número máximo de participantes
  • Valores: Inscrição, alimentação, hospedagem
4 Cadastrar Encontristas (Seguimistas)

Vincula pessoas aos encontros

  • Selecione a Pessoa
  • Selecione o Encontro
  • Informe dados de inscrição
  • Registre forma de pagamento
  • Anexe ficha de inscrição (opcional)
5 Cadastrar Tipos de Equipe

Defina os tipos de equipes que existirão

  • Exemplos: Cozinha, Limpeza, Recepção, Segurança, etc.
6 Cadastrar Equipes

Crie as equipes de trabalho vinculadas ao encontro

  • Nome da equipe
  • Tipo da equipe (cadastrado no passo anterior)
  • Descrição e responsabilidades
7 Cadastrar Trabalhadores

Atribua pessoas às equipes como trabalhadores

  • Selecione a Pessoa
  • Selecione a Equipe
  • Marque se é Dirigente
  • Informe função específica
8 Cadastrar Pastas OPCIONAL

Organize equipes em pastas para melhor hierarquia

9 Cadastrar Dirigentes

Registre dirigentes e suas pastas/equipes

10 Cadastrar Contribuições OPCIONAL

Registre contribuições financeiras dos trabalhadores/participantes

✓ Finalizar Encontro

Após conclusão do evento:

  • Edite o encontro
  • Marque como Finalizado/Inativo
  • Valide presença dos encontristas
  • Confirme valores de contribuições
  • Gere relatórios finais

Campo Tipo Obrigatório Validações

🚨 ATENÇÃO - Telefone Único:
O sistema não permite dois registros com o mesmo telefone. Esta validação evita duplicidades. Se encontrar erro ao cadastrar, verifique se a pessoa já existe no sistema.

✅ Passo a Passo do Cadastro

1 Preencha os dados básicos
  • Nome completo sem abreviações
  • Data de nascimento completa
  • Sexo
2 Adicione informações de contato
  • Telefone com DDD (campo obrigatório)
  • Email válido
3 Complete o endereço (opcional)

  • Selecione foto em JPG ou PNG

5 Adicione observações (opcional)
  • Informações relevantes sobre a pessoa
  • Restrições alimentares, medicamentos, etc.
6 Salvar

Clique em "Salvar". Uma mensagem de sucesso será exibida e você será redirecionado para a lista de pessoas.

Caminho: Menu → Casais → Novo Cadastro

O cadastro de casais permite vincular duas pessoas que formam um casal. Ambas as pessoas devem estar previamente cadastradas no sistema.

📋 Campos do Formulário

Campo Descrição
Pessoa 1 Seleção do primeiro cônjuge (campo com autocompletar)
Pessoa 2 Seleção do segundo cônjuge (campo com autocompletar)

💡 Dica: Use o campo de busca com autocompletar. Digite parte do nome e o sistema sugerirá as pessoas cadastradas. Isso facilita encontrar a pessoa correta e evita erros.
⚠️ Validação: O sistema não permite cadastrar o mesmo par de pessoas duas vezes. A ordem (Pessoa1, Pessoa2) ou (Pessoa2, Pessoa1) é considerada a mesma.

📥 Como Exportar Dados

Exportar para PDF

  1. Acesse a página do relatório desejado
  2. Configure os filtros necessários
  3. Clique no botão "Exportar PDF" ou ícone 📄
  4. O arquivo será aberto em nova aba ou baixado automaticamente

Exportar para Excel

  1. Na página de listagem, clique no botão "Exportar Excel" ou ícone 📊
  2. O arquivo .xlsx será baixado
  3. Abra com Microsoft Excel, LibreOffice ou Google Sheets
💡 Dica: Use os filtros antes de exportar para gerar relatórios personalizados. Por exemplo, filtre por um encontro específico antes de exportar a lista de encontristas.

9. Gestão de Fotos

📸 Configuração do Diretório de Fotos

O sistema armazena fotos de pessoas e encontristas em um diretório configurável no servidor.

📁 Local Padrão

Caminho padrão: C:\Desenvolvimento\fotos
(Windows)

⚙️ Configurar Caminho Personalizado

Para alterar o diretório de fotos, defina a propriedade de sistema ao iniciar a aplicação:

java -Dcaminho_fotos=/opt/segueme/fotos -jar segue-me.war
⚠️ Permissões:
Certifique-se de que o usuário do servidor de aplicação (WildFly/JBoss) tenha permissões de leitura e escrita no diretório configurado. Sem permissões adequadas, o upload falhará.

💾 Criação Automática

Se o diretório não existir, o sistema tentará criá-lo automaticamente. Verifique os logs para confirmar a criação bem-sucedida.

12. Boas Práticas

✅ Recomendações de Uso

🔐 Segurança

  • Altere senhas padrão imediatamente após primeira instalação
  • Use senhas fortes: mínimo 8 caracteres, com letras, números e símbolos
  • Não compartilhe credenciais entre usuários
  • Faça logout ao terminar de usar o sistema, especialmente em computadores compartilhados

💾 Backup e Recuperação

  • Backup regular: Configure backups automáticos do banco de dados PostgreSQL
  • Antes de grandes migrações: Sempre faça backup completo
  • Teste restaurações: Valide que os backups funcionam
  • Backup de fotos: Inclua o diretório de fotos nos backups

📋 Qualidade de Dados

  • Padronize nomenclaturas: Use "Segue-me 2026" (não "SM 2026" ou variações)
  • Evite duplicidades: Pesquise antes de cadastrar nova pessoa
  • Preencha telefones corretamente: Use formato (99) 99999-9999
  • Valide CPFs: Sempre que possível, informe CPF válido
  • Atualize status: Marque encontristas como "Presente" durante o evento

⚡ Performance

  • Use filtros: Ao trabalhar com listas grandes, aplique filtros para melhorar desempenho
  • Exporte com critérios: Não exporte tabelas completas sem necessidade
  • Limpe cache: Se encontrar lentidão, limpe cache do navegador (Ctrl+F5)

13. Solução de Problemas

🔧 Problemas Comuns e Soluções

Problema Solução
Não consigo fazer login 1. Verifique se usuário e senha estão corretos (case-sensitive)
2. Confirme que não está bloqueado (3 tentativas)
3. Contate o administrador para reset de senha
Erro ao cadastrar pessoa: "Telefone já existe" Use a busca para localizar a pessoa existente com este telefone.
Telefones são únicos no sistema para evitar duplicidades.

2. Confirme formato (JPG, PNG)

14. Perguntas Frequentes (FAQ)

❓ Dúvidas Frequentes

1. Posso cadastrar uma pessoa sem telefone?

Não. O campo telefone é obrigatório e único. Ele serve como identificador principal para evitar duplicidades.

2. Como alterar dados de um encontrista já cadastrado?

Acesse: Menu → Encontristas → Lista → Clique no ícone de edição (✏️) ao lado do registro → Modifique os campos necessários → Salvar.

3. Posso ter mais de um encontro ativo simultaneamente?

Sim. O sistema permite múltiplos encontros ativos. Use os filtros nas listas para separar dados por encontro.

4. Como marcar presença dos encontristas durante o evento?

Edite cada encontrista e marque o checkbox "Presente". Alternativamente, use relatórios para exportar lista e controlar externamente.

5. O que acontece se eu excluir um encontro?

Se houver encontristas, equipes ou outros dados vinculados, a exclusão será bloqueada. Você precisará remover todos os vínculos primeiro.

6. Posso importar dados de uma planilha Excel?

O sistema não possui importação automática via interface. Para migrações em massa, contate o administrador técnico para scripts SQL customizados.

7. As fotos ficam armazenadas no banco de dados?

Não. As fotos são armazenadas em diretório no servidor. O banco guarda apenas o nome/caminho do arquivo.

8. Como gerar etiquetas de crachás?

Exporte o relatório de encontristas em Excel e use ferramentas externas (Word, Publisher) para mala direta de etiquetas.
